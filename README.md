## Descripción
#### Este proyecto fue desarrollado como parte del Trabajo Práctico de Gestión Colaborativa.
#### El objetivo fue aplicar herramientas profesionales de desarrollo de software trabajando en equipo mediante:

    Git (control de versiones)

    GitHub (repositorio remoto)

    Jira (gestión de tareas)

    Google Colab (entorno de desarrollo)
  ###  Escenario elegido: Resultados Deportivos
  #### Se desarrolló un sistema que:

    Lee resultados de partidos desde un archivo CSV

    Procesa los datos

    Calcula puntos por equipo

    Genera una tabla de posiciones automáticamente
